### Installation  
1. Backup your old vim configuration files:  
```
mv ~/.vim ~/.vim.back
mv ~/.vimrc ~/.vimrc.back
```

2. Clone and install this repo:  
```
git clone addr ~/.vim
ln -s ~/.vim/vimrc ~/.vimrc
```
